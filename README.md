# Ardent Homepage Prototype

A Next.js prototype of the [New Ardent HP](https://www.figma.com/design/1fSszaRq7tmpuqNOHhVBdo/New-Ardent-HP) Figma design.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the prototype.

## Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Fonts: PT Sans, Archivo, Noto Sans (via Google Fonts)

## Project structure

- `src/app/` — layout, page, global styles
- `src/components/` — homepage sections (Header, Hero, Stats, Platforms, etc.)
- `public/images/` — assets exported from Figma
